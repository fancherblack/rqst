Hey look, another README!
=========================

If you're reading this, chances are you're on step #3 of the quick start installation. 

The rqst_options.csv file found in this directory is intended to establish the baseline configuration options for the app. See the README.txt in $SPLUNK_HOME/etc/apps/rqst or the app documention for more information. 

Import this file into the rqst_options KV store collection. Once imported, modify the settings according to your needs and the documentation.
